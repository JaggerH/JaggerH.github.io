[Jekyll](https://www.bilibili.com/video/BV1qs41157ZZ?p=5&vd_source=0cf989e97ecdc697d59419d1998a663f)

```bash
gem install jekyll bundler

jekyll new blog
mv blog/* ./
mv blog/.gitignore ./
rmdir blog

# 安装环境
bundle install
# 启动服务
bundle exec jekyll serve
```
